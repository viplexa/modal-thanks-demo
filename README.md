# Modal & Inline Forms Demo
Примеры модальных и inline-форм с окном «Спасибо» на jQuery

Этот репозиторий демонстрирует:

- Всплывающие модальные формы (`callback`, `feedback`)  
- Inline-формы на странице  
- Окно «Спасибо» после отправки любой формы  
- Управление overlay (закрытие по крестику, клику на фон, клавиша Esc)  
- Блокировка прокрутки страницы при открытой модалке  
- Универсальная функция `openThanksOnly()` для показа «Спасибо»
